Realm which supports user authentication with user name and password.

administrators : User (Realm) - Administrators have all permissions.

Administrators have all permissions. If administrators collection is empty, then any user is treated as a superuser. This functionality allows to configure the system after installation and then secure it by adding administrators.

                  • If there are users in the realm, the principal is not guest, and is part of the administrators reference or if the administrators reference is empty.
